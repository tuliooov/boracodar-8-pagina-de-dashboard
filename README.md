## Start project

- Add ENVs 
    - DATABASE_URL
    - FIREBASE_STORAGE_BUCKET 
    - API_KEY 
    - AUTH_DOMAIN 
    - PROJECT_ID  
    - MESSAGING_SENDER_ID 
    - APP_ID 
- npm install
- npx prisma generate
- npx prisma db seed
- npm run dev
- locale: http://localhost:3000
